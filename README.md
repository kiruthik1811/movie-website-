# 🎬 Movie Website

A modern and responsive Movie Website built using HTML, CSS, and JavaScript. It allows users to browse movies with an attractive interface and provides an enjoyable movie discovery experience.

## 🚀 Features

- 🎥 Browse movie collections
- 🔍 Search for movies
- 📱 Responsive design for mobile and desktop
- ⭐ Movie cards with posters and details
- ⚡ Fast and user-friendly interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
